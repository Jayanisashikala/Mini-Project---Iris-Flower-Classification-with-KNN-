# Iris Flower Classification with K-Nearest Neighbors (KNN)

This repository contains a mini machine learning project developed for the **CCS4340** module. The project focuses on classifying iris flowers into different species using the **K-Nearest Neighbors (KNN)** algorithm.

## 📌 Project Overview

A botanical research institute collected measurements of iris flowers and requires an automated system to classify flowers into three species:

* Iris Setosa
* Iris Versicolor
* Iris Virginica

The classification is performed using four flower measurements:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

## 🎯 Project Objectives

* Implement and evaluate a KNN classifier using the Iris dataset
* Understand how different values of **k** affect model performance
* Practice the machine learning workflow:

  * Data exploration
  * Data preprocessing
  * Model training
  * Model evaluation
* Visualize and interpret classification results

## 🤖 Machine Learning Algorithm

This project uses the **K-Nearest Neighbors (KNN)** classification algorithm.
The algorithm classifies a flower based on the nearest neighboring data points using distance calculations.
